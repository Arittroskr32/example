JWT Demo

This is a CTF challenge demonstrating a JWT vulnerability:

  • /gettoken: Issues a JWT for a user role.
    GET /gettoken
  • /admin: Protected route, requires a JWT with role: admin.
    GET /admin

Hint: What if the server doesn't validate the JWT algorithm properly?

Try forging a token and gain access to the admin panel to find the flag!
