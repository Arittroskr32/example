JWT Authentication Demo

This is a simple Flask app demonstrating JWT-based authentication:

  • /gettoken: Issues a JWT for a user role.
    GET /gettoken
  • /admin: Protected route, requires a JWT with role: admin in the Authorization header.
    GET /admin

Try to get a token and access the admin route!
Hint: The admin flag is only shown if you have a valid JWT with role: admin.
